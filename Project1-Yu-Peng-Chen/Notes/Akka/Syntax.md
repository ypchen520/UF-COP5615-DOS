# Syntax Notes
* *box*
* *x: this*

* 'a
* _
  * The mailbox is of type Actor<'a>, where 'a is the type of message the actor will handle. In most cases you can leave the type as Actor<_> and the F# compiler will infer the right message type for you.

* fun
* rec: recursive
* failwith
* match with
  * :?
  * as: give a name
* <!: send message
